## ⚒️ 安装

```shell
# pnpm
pnpm add --save-dev @fz-design/fz-design-icon

# npm
npm i --save-dev @fz-design/fz-design-icon

# yarn
yarn --save-dev add @fz-design/fz-design-icon
```
